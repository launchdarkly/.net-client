Remarks

If the result of a flag's evaluation would have returned the default variation, it will have a null entry in the map. If the client is offline, has not been initialized, or a null user or user with null/empty user key a null map will be returned.

This method will not send analytics events back to LaunchDarkly.

This method is deprecated; use AllFlagsState(User, FlagsStateOption) instead.
